# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities privately to the maintainer:

- **GitHub**: [symrex](https://github.com/symrex)

Please include the following information in your report:

- Type of vulnerability
- Affected components
- Steps to reproduce
- Potential impact
- Suggested fix (if applicable)

## Security Best Practices

- Do not commit license files or any sensitive data
- Do not expose secrets in environment variables without encryption
- Keep the Docker image updated with the latest security patches
- Use specific image tags instead of `latest` in production
